# SIGPPC

Sistema de Gerenciamento de Projetos Pedagógicos de Curso.

## Objetivo

Auxiliar na elaboração de Projetos Pedagógicos de Curso (PPC) de forma organizada, automática e baseada nos instrumentos de avaliação.

## Tecnologias

- HTML
- CSS
- JavaScript

## Estrutura

- index.html
- style.css
- app.js
- README.md

## Funcionalidades futuras

- Cadastro de PPC
- Instrumentos de avaliação
- Geração automática de relatórios
- Exportação para PDF e Word
- Controle de versões
- IA para auxiliar na elaboração dos textos
